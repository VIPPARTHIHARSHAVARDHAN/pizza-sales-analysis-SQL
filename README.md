# Business Insights from Pizza Sales Data Using SQL

## Project Overview
This project focuses on analyzing pizza sales and order data using SQL to extract meaningful business insights.  
The analysis answers real-world business questions related to revenue, customer preferences, order trends, and product performance using structured SQL queries.

---

## Objective
- Analyze sales and order data using SQL
- Identify top-selling pizzas and categories
- Understand customer ordering patterns
- Derive actionable business insights from relational data

---

## Dataset
The analysis is performed using the following tables:

- **orders** – contains order date and time information  
- **order_details** – contains pizza quantities per order  
- **pizzas** – contains pizza size and price details  
- **pizza_types** – contains pizza names and categories  

The dataset represents a real-world pizza sales order management system.

---

## Tools & Skills Used
- SQL (MySQL)
- Joins (INNER JOIN)
- Aggregate Functions (SUM, COUNT, AVG)
- GROUP BY, ORDER BY
- Subqueries
- CASE statements
- Data Analysis & Business Insights

---

## Business Questions Answered
- Retrieve the total number of orders placed  
- Calculate the total revenue generated from pizza sales  
- Identify the highest-priced pizza  
- Identify the most common pizza size ordered  
- List the top 5 most ordered pizza types along with their quantities  
- Find the total quantity of each pizza category ordered  
- Determine the distribution of orders by hour of the day  
- Analyze category-wise distribution of pizzas  
- Calculate the average number of pizzas ordered per day  
- Determine the top 3 most ordered pizza types based on revenue  
- Calculate the percentage contribution of each pizza category to total revenue  
- Analyze cumulative revenue generated over time  
- Determine the top 3 pizza types by revenue for each category

- Insights derived directly from query outputs
